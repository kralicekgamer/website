Vietcong addons

Vietcong allows you to add your own addons to the game.

  1. Download addon(link down)
  2. Unzip addon to ./addons folder
  3. To run addon use VcComHub or create shortucut with this C:\path\to\vietcong\vietcong.exe -addon [addonname]
